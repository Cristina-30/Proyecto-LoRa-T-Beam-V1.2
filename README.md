## Metodología de desarrollo incremental

Se empezó desde cero (sin conocimientos previos) y se fue avanzando paso a paso:

1. Estructura básica de Arduino  
2. Prueba de LED (validación de hardware)  
3. Transmisor LoRa básico  
4. Receptor LoRa básico  
5. Pruebas de potencia (transmisor)  
6. Receptor avanzado con RSSI, SNR y conteo de paquetes perdidos
